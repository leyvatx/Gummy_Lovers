# Gummy Lover's Frontend

React + Vite + Tailwind CSS para el dashboard operativo web.

## Desarrollo local

```powershell
npm install
$env:VITE_API_PROXY_TARGET = "http://127.0.0.1:8000"
npm run dev
```

Si Django corre en otro puerto, cambia `VITE_API_PROXY_TARGET`.

## Comandos

```powershell
npm run lint
npm run build
```
